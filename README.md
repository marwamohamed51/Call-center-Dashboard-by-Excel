# 📞 Call Center Dashboard - Excel

## 📊 Project Overview

This project presents an **interactive Call Center Dashboard** built using **Microsoft Excel** to analyze call center performance, agent productivity, and customer behavior.

The dashboard provides a clear and structured view of key performance indicators (KPIs), helping decision-makers monitor operations and improve customer service efficiency.

The **dashboard layout and UI design were first created using Figma**, then implemented in Excel to achieve a clean and modern interface.

## 🎯 Business Objectives

This dashboard helps answer important business questions such as:

* How many **calls are handled**?
* What is the **total purchase amount generated**?
* How do **agents (representatives) perform**?
* What is the **average call duration**?
* How satisfied are customers?
* How does performance change **over time (yearly trends)**?

---

## 📈 Key KPIs

The dashboard highlights key call center metrics, including:

* 📞 Number of Calls
* 💰 Total Purchase Amount
* 📊 Minimum & Maximum Purchase Amount
* ⏱ Total Call Duration
* ⏳ Average, Minimum & Maximum Duration

These KPIs are essential for tracking call center performance and improving operational efficiency ([pk-anexcelexpert.com][1]).

## 📊 Dashboard Features

### 1️⃣ Performance Overview

* Summary cards displaying key KPIs
* Quick insights into call center performance

### 2️⃣ Sales & Trends Analysis

* Total & Average Purchase Amount by Year
* Trend comparison between years

### 3️⃣ Agent Performance Analysis

* Number of Calls by Representative (R01–R05)
* Total Purchase Amount contribution by each representative

### 4️⃣ Customer Insights

* Average Satisfaction Rating by Customer
* Average Purchase Amount by Customer

### 5️⃣ Call Duration Analysis

* Total, Average, Minimum, and Maximum duration tracking
